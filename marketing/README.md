# PHALANX Top-30 Marketing Package

This folder contains the public-safe marketing deliverables produced during the Top-30 sprint on 2026-07-22.

## App Store conversion package

`app_store_conversion/top30_tier5_20260722/` contains:

- the five clean-dark, capture-ready screenshot templates;
- the five approved-layout reference composites, clearly marked as not final App Store assets;
- the clean-dark visual master and rendering script;

## Public-repository exclusions

This package intentionally excludes:

- credentials, tokens, secrets, and environment values;
- advertising-account identifiers and campaign identifiers;
- payment-account or card details;
- IPA files, app bundles, engineering-only size-audit artifacts, and app test logs;
- the 60 raw walkthrough captures and contact sheet, because they contain founder-identifying imagery and are internal QA inputs rather than final marketing assets.

Advertising, posting, website deployment, App Store submission, and spend remain separately gated. Nothing in this folder authorizes an external launch.
